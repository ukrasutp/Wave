Прежде чем подключиться к ОРС-серверу и начать коммуникации с ним клиент должен знать какой сервер ему опрашивать, 
и коль уж программное обеспечение идет по пути максимального абстрагирования отдельных программных компонентов друг от друга, 
то клиент должен иметь возможность получить информацию о всех зарегистрированных серверах предварительно ничего о них не зная.

Для решения задачи просмотра списка установленных ОРС-серверов сообществом OPC Foundation была выпущена специализированная утилита OPСEnum. 
Эта утилита находится в свободном пользовании и потратив немного времени на поиск ее можно без проблем найти в Интернете. 
Кроме того, большинство SCADA, являясь ОРС-клиентами по умолчанию, при установке инсталлируют указанную утилиту.

Для установки службы OpcEnum необходимо:
1. Скопировать файлы "opcsec_ps.dll", "opc_aeps.dll", "opcbc_ps.dll", "opccomn_ps.dll", "opchda_ps.dll", "opcproxy.dll", "OpcEnum.exe" 
   в директорию "C:\WINDOWS\System32\" для 32-разрядных систем, C:\Windows\SysWOW64\ - для 64-разрядных.
2. Запустить bat-файл "InstallOpcEnum.bat" (для 64-разрядных "InstallOpcEnum(x64).bat") для инсталяции службы.
